PHP Artisan Essential Commands: 1> php artisan make:controller controllername -> Create Controller 2> php artisan make:model modelname -> Create Model 3> php artisan make:middleware middlewarename -> Create Middleware 4> php artisan make:migration migrationname -> Create Table 5> php artisan serve -> Run Application Server 6> php artisan migrate -> To Run Migration 7> php artisan cach:clear -> Clearing Cach 8> php artisan route:clear -> Clearing Route 9> php artisan config:clear -> Clearing Config 10> php artisan view:clear -> Clearing View
